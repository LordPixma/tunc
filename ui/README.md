## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Build and Serve Public UI

1. Run `npm run build:public` to output the production build to `../public/ui`.
2. Serve the built files from `../public/ui` using any static file server, for example `npx serve ../public/ui`.
